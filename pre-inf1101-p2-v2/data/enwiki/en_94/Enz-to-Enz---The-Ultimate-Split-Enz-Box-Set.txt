Enz To Enz - The Ultimate Split Enz Box Set is an 11 CD box set by New Zealand rock band Split Enz, containing the 2006 remastered versions of the band's 9 studio albums, plus the 2006 remaster of the compilation The Beginning Of The Enz and the 2007 single disc compilation Rootin Tootin Luton Tapes. It is the band's 3rd box set, following the two released in 1993. See individual albums for track listings and credits. Category:2007 compilation albums Category:Split Enz compilation albums
